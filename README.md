# アプリの名前

## QuickDonut
https://quickdonutfb.firebaseapp.com
<br/>
業務連絡の返信が早い人ほどポイントが貯まる
<br/>
【ビジネスの信頼を可視化する】チャットアプリ
<br/>
<br/>

# 使い方・紹介動画（YouTube）
https://youtu.be/wIHEXcrCq4o
<br/>
<br/>

# このアプリでできること

* 個人間でのリアルタイムチャット
* ユーザー情報、ポイント数の事前検索
* ”即レスしてほしいメッセージ”の任意送信と、その返信によるポイント加算機能
<br/>
